## 🎯 ОСНОВНЫЕ МОДУЛИ (Core Modules)

### 1. engine.py (1,897 байт)
**Назначение:** Дифференцируемый движок для вычисления геодезических
- Автоматическое вычисление символов Кристоффеля через `torch.func.jacrev`
- Векторизация через `vmap`
- Симплектический интегратор

### 2. main.py (1,908 байт)
**Назначение:** Точка входа для базовой симуляции
- Запуск HPC симуляции
- Интеграция с logger
- Интерполяция метрики через grid_sample

### 3. logger.py (1,113 байт)
**Назначение:** Эффективное логирование в HDF5
- Потоковая запись данных
- Gzip компрессия
- Оптимизация для больших объемов

### 4. ensemble_simulator.py (19,353 байт) ⭐
**Назначение:** Ансамблевая симуляция + обратное влияние
- Класс `EnsembleSimulator` - симуляция облака частиц
- Класс `BackReactionSimulator` - с учетом влияния на метрику
- Генерация квантовой метрики через FFT
- PIC метод для тензора энергии-импульса
