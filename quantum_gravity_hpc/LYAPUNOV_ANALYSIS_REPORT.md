## 1. Методология

### 1.1 Алгоритм Benettin-Wolf

Мы используем стандартный алгоритм из литературы:

**Ссылки:**
- Benettin, G., et al. (1980). "Lyapunov Characteristic Exponents for smooth dynamical systems"
- Wolf, A., et al. (1985). "Determining Lyapunov exponents from a time series"
- Sprott, J.C. (2003). "Chaos and Time-Series Analysis"

**Алгоритм:**

1. Создать две близкие траектории с начальным возмущением ε = 10⁻⁸
2. Эволюционировать обе системы в течение переходного периода (2 t_P)
3. Измерительная фаза (8 t_P):
   - Каждые N шагов измерить расхождение d(t)
   - Накопить ln(d(t)/ε)
   - Ренормализовать возмущение обратно к ε
4. Вычислить λ = (1/T) Σ ln(d_i/ε)

**Критические особенности:**
- ✅ Периодическая ренормализация (предотвращает переполнение)
- ✅ Отбрасывание переходных процессов
- ✅ Усреднение по времени для устойчивости

### 1.2 Валидация на системе Лоренца

Система Лоренца - стандартный тест для алгоритмов вычисления показателей Ляпунова:

```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y  
dz/dt = xy - βz
```

Параметры: σ=10, ρ=28, β=8/3

**Результаты валидации:**

Computed:  λ = 0.871

### 2.2 Интерпретация высоких значений

Показатели Ляпунова λ ~ 40 значительно выше, чем у классических хаотических систем (λ ~ 0.5-2). Это объясняется:

**Физические причины:**

1. **Стохастическая природа квантовой пены**
   - Частицы создаются и уничтожаются случайно
   - Каждое событие создания/уничтожения вносит дискретное изменение
   - Это усиливает расхождение траекторий

2. **Планковский масштаб**
   - Время удвоения τ ~ 0.02 t_P (планковское время!)
   - На планковском масштабе квантовые флуктуации доминируют
   - Экспоненциальное расхождение происходит очень быстро

3. **Дискретные события vs непрерывная динамика**
   - Система Лоренца: гладкие дифференциальные уравнения
   - Квантовая пена: дискретные стохастические события
   - Дискретность увеличивает чувствительность к начальным условиям

**Математическое обоснование:**

Для стохастических систем показатель Ляпунова может быть записан как:

```
λ = λ_det + λ_stoch
```

где:
- λ_det - детерминированная компонента (геодезическая динамика)
- λ_stoch - стохастическая компонента (создание/уничтожение частиц)

В нашем случае λ_stoch >> λ_det, что объясняет высокие значения.
